Angular wrapper for Jed

See the Github page for the complete documentation

The module provides a filter and a directive for all kinds of translations.

Translations in controllers

{{ title }}

// controller
app.controller('HomeController', [
  '$scope', 'i18n', function($scope, i18n) {
    // load a page translation
    i18n.loadPage('home').then(function() {
      $scope.title = i18n._('Text translated from the controller');
    });
]);
<!-- template -->
<p>{{ title }}</p>
